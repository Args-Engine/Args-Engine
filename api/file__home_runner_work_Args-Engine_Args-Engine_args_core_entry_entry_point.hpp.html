Detailed Description¶

When defining ARGS_ENTRY do no create your own CRT_STARTUP such as main()->int, main(int,char**)->int,wmain(), etc… When using ARGS_ENTRY you must instead implement Function reportModules(args::core::Engine *). When not using ARGS_ENTRY you must call creation and initialisation of the engine manually.
