class args::core::Engine¶

Main top level engine abstraction. This class allows you to setup the engine with all the necessary modules and settings. This class also contains all the sub managers of the core systems. args::core::Module args::core::ecs::EcsRegistry args::core::scenes::SceneManager args::core::scheduling::Scheduler.
