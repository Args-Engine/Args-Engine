LEGION_HIGH_PERFORMANCE¶

Automatically defined if LEGION_LOW_POWER was not defined. It makes Legion ask the hardware’s full attention to run as fast as possible.

Note

Define LEGION_LOW_POWER to run Legion with minimal resources instead.
