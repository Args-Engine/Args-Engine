struct legion::audio::audio_import_settings¶

Import settings for audio files.

Settings: force_mono: when enabled the loaded audio file will combine channels to make the audio file mono, which allows for spatial audio split_channels: when enabled the channels of the audio file will be loaded into seperate audio segments
