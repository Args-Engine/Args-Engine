class args::core::async::readonly_guard¶

RAII guard that uses ::async::readonly_rw_spinlock to lock for read-only.

Note

Read-only operations can happen simultaneously without waiting for each other. Read-only operations will only wait for Read-Write operations to be finished. args::core::async::readonly_rw_spinlock
